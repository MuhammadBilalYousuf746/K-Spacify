# K-Spacify — Local Community Hall & Ground Booking

A simple platform to book local venues in Karachi — halls, lawns, and turfs for events like birthday parties, dholkis, and cricket matches.

## About

Finding and booking local venues in Karachi is a hassle — you have to visit each place in person just to check rates and availability. **K-Spacify** solves this in one place.

### Core Features

- **Venue Explorer** — Browse venues filtered by area and price (e.g., "Halls under 50,000 PKR in North Nazimabad")
- **Slot Availability** — See which dates are booked and which are open on a simple calendar
- **Booking Request** — Submit a booking request online; admin can then Approve or Reject it
